
# Git Utilities

Usefull utilities for GIT users

### git-gen-rep
A release report generator:
  * collects udpates in release_notes.txt (or ReleaseNotes.txt)
  * collects updates on source code changes.txt
  * collects GIT commits statistics

### pre-commit
A precommit hook to:
  * get rid of tab symbols, trailig whitespaces
  * support Perforce keywords
